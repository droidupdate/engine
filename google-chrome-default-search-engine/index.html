If you want to change the default search engine in Google Chrome, you can do so by following these steps:

  1. Open Google Chrome.
  2. Click on the three dots in the top-right corner of the browser window.
  3. Select "Settings" from the drop-down menu.
  4. Scroll down to the "Search engine" section.
  5. Select your desired search engine from the drop-down menu.
  6. Click on the "Save" button.
